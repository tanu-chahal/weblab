üîß Troubleshooting Notifications:

If notifications aren't showing:
  1. Check browser settings: Make sure notifications aren't blocked globally
  2. Site permissions: Click the lock/info icon in address bar ‚Üí Allow notifications
  3. Focus issue: Try switching to another tab/app - some browsers only show notifications when page isn't focused
  4. macOS: Check System Preferences ‚Üí Notifications ‚Üí Your Browser
  5. Windows: Check Settings ‚Üí System ‚Üí Notifications & actions
  6. Do Not Disturb: Make sure Do Not Disturb mode is off
For Push API / Service Worker issues:
  1. Serve over HTTP/HTTPS: Service Workers don't work with file:// protocol
  2. Use a local server: Try Live Server extension in VS Code or python -m http.server
  3. HTTPS required: Push API requires HTTPS in production (localhost works with HTTP)
  4. Check console: Open Developer Tools to see detailed error messages
Focus/Background notification behavior:
  1. Browser Focus Policy: Most browsers only show notifications when the tab/window is NOT active
  2. Test Methods: Use "Force Notification" button for guaranteed results
  3. Real Focus Test: Click "Test Focus Change", then switch to another app/tab and come back
  4. Visibility API: Modern browsers use Page Visibility API to detect when tabs are hidden
  5. Desktop vs Mobile: Behavior may differ between desktop and mobile browsers
