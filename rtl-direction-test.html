Testing Instructions

How to Test --force-ui-direction Flag

  1. Open Chrome/Chromium browser
  2. Start with flag: --force-ui-direction=rtl
  3. Load this page and observe layout changes
  4. Check that all elements mirror correctly
  5. Test with --force-ui-direction=ltr for comparison
  6. Use the toggle button above to manually switch directions
